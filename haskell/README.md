|  **Comando** | **Signficado**  |
|---|---|
|  :load | Carrega um arquivo  |
|  :reload | Recarrega modificações  |
|  :edit | Edita arquivo atual  |
|  :type <expr> | Mostra o tipo de uma execução  |
|  :help | Ajuda  |
|  :quit | Encerra a sessão  |
|  :browse <module> | Mostra todas as funcões de um módulo  |
